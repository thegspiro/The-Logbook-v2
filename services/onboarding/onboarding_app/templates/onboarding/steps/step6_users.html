Note: You can add users after completing the onboarding process through the admin panel. This step helps you plan your user structure.

Available User Roles

Administrator

Full system access. Can manage users, configure settings, and access all modules.

Officer

Can create and edit incident reports, manage rosters, view analytics, and coordinate activities.

Firefighter

Can view schedules, access training materials, update availability, and view basic reports.

Support Staff

Limited access for administrative support, billing, and auxiliary functions.

Planning Your User Structure

Consider the following when planning user accounts:

  • How many administrators do you need? (Recommend 2-3 for redundancy)
  • Will officers need different permission levels?
  • Should all firefighters have immediate access, or invite-only?
  • Do you need separate accounts for training coordinators or equipment managers?

You'll be able to create the first administrator account and invite other users after completing this onboarding.
